# 《高中的那些没用的小知识》

这是一本 *没什么用* 的小书，旨在探讨并扩展一些高中知识。

本书使用 $\rm{\LaTeX}$ 编写。

## 本地编译

使用`xelatex`编译

```bash
xelatex main.tex
```
